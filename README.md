# ToDo List Frontend

## Stack:
- TypeScript
- Next.js
- Tailwind CSS
- Axios

## Instructions to Start the Project in Dev Mode:
1. Clone the repository.
2. Navigate to the project directory.
3. Install dependencies using `npm install`.
4. Run the development server using `npm run dev`.
